ATM Interface

Overview-->
This is a simple Java ATM Interface application built using JavaFX.
It allows users to perform basic banking operations like Deposit, Withdraw, and Check Balance in a graphical user interface.


Features-->
Deposit money into the account
Withdraw money from the account
Check Balance
User-friendly JavaFX interface with buttons, labels, and text fields
Real-time updates to balance
 
Technologies Used-->
Java 
JavaFX (UI Framework) 

Setup & Run-->
1. Clone the repository
git clone https://github.com/your-username/ATM-Interface.git
cd ATM-Interface

2.Install JavaFX
Make sure you have JavaFX SDK installed and set up.
You can download it from: https://openjfx.io/

3.Compile & Run
Example (using terminal):
javac --module-path "path/to/javafx-sdk/lib" --add-modules javafx.controls src/*.java
java --module-path "path/to/javafx-sdk/lib" --add-modules javafx.controls Main

Usage-->
Deposit: Click the Deposit button, enter an amount, and press Enter.
Withdraw: Click the Withdraw button, enter an amount, and press Enter.
Check Balance: Displays the current account balance.

Future Improvements-->
